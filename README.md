# 🛡️ Virtual Home Lab (Windows + Kali Linux)

## 📌 Overview

This project documents the setup of a personal cybersecurity home lab built on a Mac host machine. The lab simulates a real-world environment with:

- A Windows virtual machine (target system/defence)
- A Kali Linux VM (attacker machine)

The goal is to practice penetration testing, vulnerability assessment, and defensive monitoring in a safe, isolated environment.

## 🎯 Objectives

- Build a virtualised lab environment on macOS
- Configure a secure internal network between VMs
- Perform basic offensive security testing using Kali Linux
- Analyse vulnerabilities on a Windows system
- Develop hands-on skills aligned with CompTIA Security+ and CySA+

## 🧰 Technologies Used

- Host OS: macOS
- Virtualisation: UTM
- Guest OS 1: Windows 11
- Guest OS 2: Kali Linux

## Tools:

- Splunk
- Wireshark
- Nmap
