# Screenshots

Below are step-by-step screenshots to help guide you through setting up your Kali Linux VM using UTM.

---

## Step 1: Create a New Virtual Machine

<img width="898" height="646" alt="Create New VM" src="https://github.com/user-attachments/assets/99aaf162-0ccc-49e6-9f65-8ae75d64ae23" />

Within UTM, select **"Create a New Virtual Machine"** to begin the setup process.

---

## Step 2: Select Your Operating System

<img width="448" height="512" alt="Select Windows OS" src="https://github.com/user-attachments/assets/0359405b-3c32-47fa-b703-518dfec1ee76" />

You will then be asked which operating system you would like to install.  
For this demonstration, I selected **Linux**.

---

## Step 3: Add Your Windows ISO File

<img width="450" height="502" alt="Add ISO File" src="https://github.com/user-attachments/assets/a6c590c8-b123-4c1c-8685-dad8fc4781cd" />

At this stage, UTM will request an ISO file.  
This file is used to install and boot the operating system inside your virtual machine.

---

## Step 4: Download the Kali Linux ISO

<img width="1347" height="805" alt="Screenshot 2026-04-24 at 17 27 40" src="https://github.com/user-attachments/assets/ee5f83cc-0921-4551-a09a-d438a8a21879" />

You can download the official Kali Linux ISO directly from Microsoft here:

🔗 https://www.kali.org/get-kali/#kali-installer-images

---

## Step 5: Launch the Virtual Machine

<img width="904" height="649" alt="Screenshot 2026-04-24 at 17 51 07" src="https://github.com/user-attachments/assets/dafc8c18-6870-4e8b-b2d3-f74a847c0ac0" />

Once the ISO has been successfully added, your UTM configuration should display a **Play button**.


Select this to start your Windows VM.

---

## Step 6: Complete Setup and First Boot

<img width="1229" height="801" alt="Screenshot 2026-04-24 at 17 53 35" src="https://github.com/user-attachments/assets/164b07fc-3f04-46f0-9d26-b23d4c408592" />

The VM will complete several setup stages, including updates and initial configuration options.

After this process finishes, your new Kali Linux Virtual Machine will load to the main desktop environment and be ready for use.

---

## Final Result

<img width="1277" height="806" alt="Screenshot 2026-04-05 at 12 59 33" src="https://github.com/user-attachments/assets/43329076-e29f-4489-886c-c56c57956744" />

You now have a fully functioning Kali Linux VM running inside UTM, ready for testing, security labs, and hands-on learning.
