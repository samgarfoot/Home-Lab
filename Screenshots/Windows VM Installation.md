# Screenshots

Below are step-by-step screenshots to help guide you through setting up your first Windows VM using UTM.

---

## Step 1: Create a New Virtual Machine

<img width="898" height="646" alt="Create New VM" src="https://github.com/user-attachments/assets/99aaf162-0ccc-49e6-9f65-8ae75d64ae23" />

Within UTM, select **"Create a New Virtual Machine"** to begin the setup process.

---

## Step 2: Select Your Operating System

<img width="448" height="512" alt="Select Windows OS" src="https://github.com/user-attachments/assets/0359405b-3c32-47fa-b703-518dfec1ee76" />

You will then be asked which operating system you would like to install.  
For this demonstration, I selected **Windows 11**.

---

## Step 3: Add Your Windows ISO File

<img width="450" height="502" alt="Add ISO File" src="https://github.com/user-attachments/assets/a6c590c8-b123-4c1c-8685-dad8fc4781cd" />

At this stage, UTM will request an ISO file.  
This file is used to install and boot the operating system inside your virtual machine.

---

## Step 4: Download the Windows 11 ISO

<img width="1278" height="838" alt="Windows ISO Download" src="https://github.com/user-attachments/assets/fcedeb23-85b3-409b-9eb4-334765bb4430" />

You can download the official Windows 11 ISO directly from Microsoft here:

🔗 https://www.microsoft.com/en-us/software-download/windows11

---

## Step 5: Launch the Virtual Machine

<img width="896" height="656" alt="Play Button VM" src="https://github.com/user-attachments/assets/8dfa5f13-a333-490f-8831-069c63f9613f" />

Once the ISO has been successfully added, your UTM configuration should display a **Play button**.

Select this to start your Windows VM.

---

## Step 6: Complete Setup and First Boot

<img width="1027" height="769" alt="Windows VM Setup Complete" src="https://github.com/user-attachments/assets/36bdef16-5736-4cb9-a52e-b457c85e68cd" />

The VM will complete several setup stages, including updates and initial configuration options.

After this process finishes, your new Windows 11 Virtual Machine will load to the main desktop environment and be ready for use.

---

## Final Result

You now have a fully functioning Windows 11 VM running inside UTM, ready for testing, security labs, and hands-on learning.
